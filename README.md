1. ğŸ“Š Slide Deck Builder (Images to PowerPoint)

A simple tool to convert a batch of screenshots or images into a formatted PowerPoint presentation (.pptx) instantly.

Features:

Drag & Drop: Upload multiple images at once.

Auto-Sort: Automatically sorts files by name (e.g., slide-1.jpg, slide-2.jpg).

Manual Reordering: Move slides around if the auto-sort isn't perfect.

Instant Export: Generates a standard Widescreen (16:9) PowerPoint file.

Tech Stack: HTML5, Tailwind CSS, PptxGenJS.
