### 1. üìä Slide Deck Builder
Convert a batch of images into a professional PowerPoint presentation instantly. Perfect for designers, developers, and students who need to stitch screenshots into a slide deck.

## ÔøΩ Getting Started

Since these tools are purely static, you can run them on any local web server or even just by opening the file.

### Prerequisites
*   A modern web browser (Chrome, Edge, Firefox, Safari).
*   No Node.js or backend required.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/notsaqlain/web-tools.git
    ```

2.  **Navigate to the directory**
    ```bash
    cd web-tools
    ```

3.  **Run the App**
    *   Simply double-click `index.html` to open it in your browser.
    *   *Optional:* Use a lightweight server like `live-server` or `python -m http.server` for the best experience.
